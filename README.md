# MSCS532 Assignment 1 - Insertion Sort (Decreasing Order)

## Overview
This repository contains a Python implementation of the Insertion Sort algorithm that sorts an array in monotonically decreasing order (largest to smallest).

## How to Run
1. Install Python 3.8+
2. Run the program:

```bash
python3 insertion_sort_desc.py
```

## Example Output
Original Array: [5, 2, 9, 1, 5, 6]  
Sorted Array (Decreasing): [9, 6, 5, 5, 2, 1]

## Reference
Cormen, T. H., Leiserson, C. E., Rivest, R. L., & Stein, C. (2022). *Introduction to Algorithms* (4th ed.). Random House Publishing Services.
